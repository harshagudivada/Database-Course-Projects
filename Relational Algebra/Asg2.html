Assignment 2: Evaluation of Relational Algebra Expressions over Files

Assigned in course CS3011: Database Systems Lab

This assignment can be done in groups of 'at most' two members. The assignment has two important parts: (1) implementing the 6 basic relational algebra operators and (2) implementing an expression evaluator to evaluate relational algebra expressions. It is recommended that you take a step-by-step approach.  Your work for each step should be separately testable.


First, implement the six basic relational algebra operators for relations stored as delimited text files. Each operator must be implemented as a separate C++ class. All operator classes will extend an abstract base class (in other words implement the interface) 'RAOperator'. Define a suitable virtual function in the 'RAOperator' class, which is overridden by the concrete operator classes to implement their functionality. The inheritence hieararchy can have other classes to suitably model different classes of operators (e.g., BinaryRelationalOp and UnaryRelationalOp).


 


Using these operators, implement an evaluator for relational algebra expressions. The expression syntax is specified below through examples.

Example 1:
project<account_number, balance>(select<account.branch_name=branch.branch_name ^ branch_city='Oslo' ^ balance > 10000> (account X branch))

Example 2:
project<balance>(account) MINUS project<account.balance> (select<account.balance < a.bal> (account X rename<a(an, bn, bal)> (account)))

Each relation r will be stored in a file named r.dat, using the following format:
Row separator: newline, column/field separator: pipe (|)
The first row contains column metadata, in the format attr_name:type.

Sample contents of account.dat

account_number:int|balance:float|branch:string
1001|24000.5|Kukatpally
2004|56010.0|Andheri
1008|2000|Koramanagala
1055|3050|Hauz Khas

You need to support only 3 data types - int, float and string.

The expression evaluator should produce its output on stdout, in the same format as the base relation files.



To implement the second step of this assignment you will need to learn Lex and Yacc:
